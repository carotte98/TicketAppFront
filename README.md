# TicketWebsite

Small Ticketing App project made using Angular and an API made in .Net. 

## Dependencies to start
- API : https://github.com/carotte98/TicketAppAPI
- AngularCLI
- NPM

## Project dependencies used

- PrimeNg, PrimeNgIcons
- Tailwind V3
- Angular

## Starting the project

- Follow the instructions on the API project to start the 
- Clone the repository locally
- In terminal (right click : Start terminal here, or in your IDE) : npm install
- When everything is installed, in terminal: ng serve
- Click the link in the terminal

## Development server

To start a local development server, run:

```bash
ng serve
```

## Building

To build the project run:

```bash
ng build
```
